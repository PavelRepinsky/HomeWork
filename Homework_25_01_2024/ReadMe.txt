ParallelFactorialCalculator v0.1

Выполненное задание представлено двумя файлами:
1. ParallelFactorialCalculator - Приложение, которое высчитывает факториал каждого из имеющихся числе в файле numbers.txt
В программе под расчёты факториалов выделяется 5 потоков, при необходимости это число можно увеличить (нужно увеличить цифру в Executors.newFixedThreadPool(X))

2. Файл numbers.txt, который содержит 50 чисел (вместо рандомных сделал просто по порядку)
