CachedProxyApp v0.1

Annotations: 
1. Cached - аннотация указывает на то, что результат работы метода нужно кешировать;
2. Limit-Size - наличие этой аннтоации говорит о том, что кешируем только указанное количество элементов List (в программе используется ArrayList)

Interfaces:
1. HardWork - интерфейс, содержащий метод doHardWork() (возвращает String)
2. ReturnList - интерфейс, содержащий метод returnedList() (возвращает ArrayList<String>)

Classes:
1. CachedProxyApp - Основное приложение
2. CacheSettings - класс отвечает за настройки кеша
3. FileToZip - класс используется для упаковывания результата работы метода в zip архив
4. ListTrimer - работа класса заключается в обрезке списка, который мы кешируем
5. TrimListException - исключение, бросаемое при значении переменной amount < 0 (во время обрезки списка)

Папка SaveSuda содержит артефакты сериализации результатов работы методов и для примера zip архив с результатом работы метода doHardWork()
